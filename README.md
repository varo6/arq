## trabajo arq

La idea es hacer como un filtrador de palabras, donde la maquina está escuchando letra por letra. Enlazando cada letra en una maquina de estados va comparando si al final lo que se escribe mediante la maquina es correcto. EJEMPLO: "CASA" -> Compara si C -> estado del 0 al 1, si A -> del 1 al 2, si S -> 2 al 3 pero si ahora no es S -> volvemos al estado 0.

## Posibles perifericos a añadir


- Memoria: RAM o lo que sea para almacenar la palabra a filtrar
- VGA: Periferico y por ej mostramos verde o rojo según si se ha escrito la palabra
